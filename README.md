## ✨ Основной функционал

* 🍕 **Каталог товаров:** Удобный просмотр меню и добавление позиций в корзину.
* 🧺 **Умная корзина:** Возможность проверить свой заказ, удалить лишнее и увидеть итоговую сумму перед оформлением.
* 👤 **Профиль пользователя:** Автоматический подсчет количества заказов и общей суммы потраченных средств.
* 🚀 **Оформление заказа:** Быстрая отправка заказа в базу данных с автоматической очисткой корзины.
* 🔔 **Админ-панель:** Мгновенные уведомления администратору/менеджеру о новых заказах с детальной информацией о покупателе.

## 🗄 Структура базы данных

Бот использует реляционную структуру из 4 основных таблиц:

* `users` — хранит данные пользователей и их статистику *(потрачено денег, всего заказов)*.
* `products` — каталог товаров с актуальными ценами.
* `cart` — временное хранение товаров до момента оформления.
* `orders` — история подтвержденных заказов.
